# Калькулятор (Expo / React Native)

Мобільний додаток-калькулятор. Залікова робота — **Варіант 7**.

## Функціонал

- Базові арифметичні операції: `+`, `-`, `×`, `÷`
- Підтримка десяткових дробів (кнопка `.`)
- Обробка помилок: ділення на нуль виводить слово **«Помилка»**
- Додаткові функції: `%` (відсоток), `⌫` (видалення останнього символу), `C/AC` (часткове / повне очищення)
- Ланцюжкові обчислення з підсвічуванням активного оператора
- Превʼю поточного виразу над основним результатом
- **Історія обчислень** у модальному вікні: список усіх виконаних операцій, тап на запис підставляє результат назад у дисплей, кнопка «Очистити» прибирає всю історію

## Стек

- [Expo](https://expo.dev/) (шаблон `blank`)
- React Native (`react`, `react-native`)
- JavaScript (без TypeScript)
- Стандартні компоненти RN: `SafeAreaView`, `View`, `Text`, `Pressable`, `Modal`, `FlatList`

## Структура проєкту

```
calculator-app/
├── App.js          # Уся логіка та UI калькулятора
├── app.json        # Конфігурація Expo
├── index.js        # Точка входу (registerRootComponent)
├── package.json    # Залежності та скрипти
├── assets/         # Іконки та splash
└── README.md
```

Уся логіка зосереджена в одному файлі `App.js`:

- Чиста функція `calculate(a, b, op)` — повертає рядок-сентинел `'Помилка'` при діленні на нуль.
- Функція `formatNumber(num)` — округлює результат до 10 знаків після коми, прибираючи артефакти плаваючої точки (наприклад, `0.1 + 0.2`).
- Стани через `useState`: `display`, `previous`, `operator`, `waitingForOperand`, `history`, `historyVisible`.

## Запуск

### 1. Встановлення залежностей

```bash
cd calculator-app
npm install
```

### 2. Запуск Expo dev-сервера

```bash
npm start
```

### Запуск на телефоні через Expo Go

1. Встановіть на телефон додаток **Expo Go** (Google Play / App Store).
2. Виконайте `npm start` у папці проєкту.
3. Відскануйте QR-код, що зʼявиться в терміналі.

## Сценарії перевірки

| Дія                   | Очікуваний результат |
| --------------------- | -------------------- |
| `2 + 3 =`             | `5`                  |
| `7 × 8 =`             | `56`                 |
| `10 ÷ 4 =`            | `2.5`                |
| `0.1 + 0.2 =`         | `0.3` (без хвоста)   |
| `5 ÷ 0 =`             | `Помилка`            |
| `12 → C`              | `0` (часткове очищення); друге натискання → `AC` (повний скид) |
| Тап на запис в історії | Підставляє результат у дисплей |
